# dds

DSS 웹사이트는 개발자들 스터디모집 사이트 입니다

<br>
개발 기간 : 2022-07-13 ~ 

## :heavy_check_mark:사용기술 

언어 
* JAVA
* HTML / CSS / JAVASCRIPT
* JQUERY

웹서버
* TOMCAT

데이터베이스
* HEIDISQL

디자인 패턴 
* MVC 적용

## :pushpin:주요기능 

* 회원 기능 ( 회원가입, 로그인, 로그아웃,... )
* 모집 게시판 기능 ( 게시판 등록, 수정, 삭제,.. )
* 알림 기능 ( 모집 등록 알림,.. )
